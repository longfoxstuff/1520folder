Projects for CIST 1520


Project 1 - Write code to do something

Create an HTML file with included javascript to let the user enter a series of class test scores. Then, when the user enters any negative number instead of a valid score, the application ends the series and displays the average test score.

Use some kind of looping and use the built-in prompt() function. The value returned from prompt() is a string, so use parseInt() to convert it to a numeric. Test for valid scores. Valid scores range from 0 to 100.

The formula for average score is: total_of_all_scores / number_of_scores; The javascript in your loop should accumulate the values for these two variables. Display your calculated average on the screen using innerHTML.
